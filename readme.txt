Technical specification

Core concept: 
The game is a set of identical cards. The goal of the game is to reveal cards in pairs. 
When the cards match, they are removed.The task is to completely clear the playing field.

At this stage the game has implemented:
Cards rendered to screen with Java Script. The cards are first turned over to the side with the picture, 
and when pressed again, it return to original state. The card turns over with an animation effect
